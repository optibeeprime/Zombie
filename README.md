Zombie FPS 
制作時期: 2025年12月～2026年1月

ゲーム概要

本作は FPS視点で銃を使い、ゾンビを倒しながら脱出を目指すゲームです。
プレイヤーはマップ内を探索し、ゾンビを倒して鍵の破片を集め、
最終的に出口へ向かうことでクリアとなります。

ゲームの流れ
	1.	FPS視点で探索
	•	一人称視点（FPS）でマップを移動します。
	•	銃を使ってゾンビと戦います。
	2.	ゾンビを倒して鍵の破片を入手
	•	ゾンビを倒すと、最大で 3個の鍵の破片 がドロップします。
	•	さらに、マップ内には あらかじめ隠された鍵の破片が2個 存在します。
	•	合計で 5個の鍵の破片 を集めることができます。
	3.	合成台で鍵を作成
	•	ミニマップ上の 黄色の点 が合成台の位置です。
	•	鍵の破片を5個集めると、合成台で 1つの鍵 を合成できます。
	4.	出口へ向かう
	•	ミニマップ上の 緑の点 が出口です。
	•	鍵を持った状態で出口に到達すると ゲームクリア になります。


ミニマップの表示
	•	🟡 黄色の点：合成台
	•	🟢 緑の点：出口

ミニマップを頼りに目的地を確認しながら進みましょう。

操作方法

キー	操作
W / A / S / D	移動
Space	ジャンプ
E	合成台で鍵を合成
Z	スタック時用リスポーン
F	死亡時リスポーン

クリア条件
	•	鍵の破片を 合計5個 集める
	•	合成台で 鍵を1つ作成
	•	出口（ミニマップの緑の点）に到達する
